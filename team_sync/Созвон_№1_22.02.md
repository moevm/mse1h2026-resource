# Протокол встречи

## Обсуждение возможности реализовать универсальный сбор данных телеметрии
**Проблема** - данные телеметрии уникальны для различных сервисов(например разные СУБД обладают разными метриками)

**Предлагаемое решение** - создать собственную спецификацию, для входных данных, переложив на пользователя ответсвенность за соблюдение формата

**Необходимо вынести вопрос на обсуждение с заказчиком**
